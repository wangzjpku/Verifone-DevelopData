========================================================================
       FRAMEWORK : LAB
========================================================================


VerixV AppWizard  has created the LAB app and uses the rudimentary 
features of the Framework and is also the starting point for adding your required 
functionality.

This file contains a summary of what you will find in each of the files that
make up your application.


LAB.c
	Starting point of the application. Contains the basic initialization and a sample 
	
LAB.smk
        This is the NMAKE file for building the .out file for the VerixV terminal

d.bat
        This is the batch file for downloading the required files to the VerixV terminal.

LAB.dld
	dld file lists the required files to be downloaded into the terminal.


LAB.fst
        This fst file gives info for signing the out file.
